# Hardware Connection Guide (Raspberry Pi)

## Ultrasonic Sensors

### Ultrasonic Sensor Front Connections
| Pin | Connection |
| :--- | :--- |
| VCC | 3.3V |
| Trig | GPIO 23 |
| Echo | GPIO 24 |
| GND | GND |

### Ultrasonic Sensor Left Connections
| Pin | Connection |
| :--- | :--- |
| VCC | 3.3V |
| Trig | GPIO 17 |
| Echo | GPIO 27 |
| GND | GND |

### Ultrasonic Sensor Right Connections
| Pin | Connection |
| :--- | :--- |
| VCC | 3.3V |
| Trig | GPIO 5 |
| Echo | GPIO 6 |
| GND | GND |

---

## Wrist Button Pad Connections

| Button | Connection |
| :--- | :--- |
| **Button 1** (Describe Scene) | GPIO 16 to GND |
| **Button 2** (Read Text) | GPIO 20 to GND |
| **Button 3** (Toggle Nav Mode) | GPIO 21 to GND |
| **Button 4** (Repeat Last) | GPIO 12 to GND |

---

## Speaker & Camera Connections
- **Speaker**: Direct to 3.5mm Audio Jack.
- **Camera**: Ribbon cable direct to Raspberry Pi CSI Port.
